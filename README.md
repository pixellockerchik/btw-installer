# Скрипт для быстрой установки и настройки bspwm, sxhkd, polybar
Данный скрипт предназначен для личного использования, но никто не запрешает запускать его и на вашем устройстве ;). Это мой первая программа которую я написал на языке shell.
# Установка
Чтобы установить и сразу запустить программу выполните команду:
```
git clone https://github.com/pixellockerchik/btw-installer && cd btw-installer
sh ./installer.sh
```
